Atributos

Apuntes

Los formularios son utilizados para que los usuarios ingresen sus datos para subscribirse a un sitio o para contactar con la empresa encargada del sitio.

Los formularios en HTML se crean con la etiqueta "form". Dentro de la etiqueta form se encuentran los atributos "action" y "method". Action sirve para mandar los datos capturados del formulario, el valor tiene que tener el enlace al archivo PHP. Method sirve para especificar el método de envío.

La etiqueta label se usa para asociar un texto con un elemento de formulario. El atributo for se utiliza para especificar el ID del elemento de formulario con el que el label está relacionado.

El atributo "name" dentro de la etiqueta label debe coincidir con el utilizado en PHP para que este funcione.

Input se utiliza para crear un campo de entrada, este se define con el atributo "type".

Los tipos de input incluyen: text, password, email, number, date, time, file, checkbox, radio, submit, reset, range, color, tel, url y search.

La etiqueta text area define un área de texto para introducir varias líneas de texto.

La etiqueta button sirve para realizar acciones en el formulario, ya sea enviar o borrar.

La etiqueta select sirve para poner un menú desplegable en donde se encuentran varias opciones.

El atributo ID es un identificador, tiene que ser único en todo el documento.

El atributo value es el valor inicial dentro del campo.

El atributo placeholder es un texto de ayuda para el usuario

El atributo required hace obligatorio al campo

El atributo disabled desactiva el campo

El atributo maxlenght determina un numero maximo de caracteres permitidos

Los atributos cols y rows limitan el numero de caracteres por columnas y renglones en el textarea.

Resumen
