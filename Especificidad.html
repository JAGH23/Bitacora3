Regla !important

Apuntes

Es el medio por el cual los navegadores deciden que valores de la propiedad css son los mas relevantes para un elemento y por lo tasnto se aplicaran.

Solo se aplica cuando el mismo elemento es objetivo de varias declaraciones.

Cada selector tiene su lugar en la jerarquía de especificidad. Hay 4 categorías que definen el nivel de especificidad de un selector.

1. Estilos inline: Los elementos style que se asignan en el html de manera directa. Tiene especificidad de 1000.

2. ID's: Si el elemento html posee un ID y se hace referencia a este en el CSS. Tiene especificidad de 100.

3. Clases, atributos y pseudoclases. Tiene especificidad de 10.

4. Selector universal. Tiene especificidad de 0.

La regla !important anula cualquier otra declaración, solo se utiliza para arreglar estilos rápidamente. Aunque !important no tenga nada que ver con la especificidad, interactua directamente con ella.
