Apuntes

Permiten insertar contenido en una pagina web desde CSS sin usar el HTML. Si bien no está en el documento, si aparece en la página. Se debe usar la propiedad "content" para especificar el contenido que se insertará.

El valor de la propiedad content puede ser una cadena de carácteres (content:"texto";), imágenes(content:url(imagen.jpg);) o nada (content:"";)

En texto los caracteres especiales deben estar especialmente codificados como una entidad unicode.

En las imágenes no se altera el tamaño, por lo que se muestra el tamaño original.

El content vacío resulta útil para insertar formas.

El pseudo-elemento ::after sirve para insertar algo después de los elementos seleccionados.

El pseudo-elemento ::before sirve para insertar algo antes de los elementos seleccionados.

Resumen
