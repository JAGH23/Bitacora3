Importancia del header

Elementos del header

Elementos de tienda online

Apuntes

Es la parte más importante puesto que es lo primero que ve el usuario. Es un espacio genérico que tiene el mismo diseño en todas las páginas del sitio web.

Los elementos que pueden formar parte del header van a depender de los objetivos del sitio web. Logotipo, menu, buscador, datos de contacto, redes sociales, carrito de compra, etc.

No puede faltar el logotipo, puesto que este enlaza la página principal y reforzar la imagen de la marca.

Se puede colocar un menú o dos dependiendo del proyecto. Si se usaran muchos enlaces en el nav, lo mejor es que no esté en el header.

El buscador es por cuestiones de usabilidad, un usuario que entra por primera vez al sitio usará esta herramienta para buscar contenido. El buscador debe usar la base de datos de la pagina web y tiene que estar dentro de la pagina.

Los datos de contacto que se pueden incluir son pocos, como telefono o e-mail.

Si se trata de una tienda online, se deben incluir los elementos referentes al proceso de compra, como un carrito de compra, lista de deseos, cuenta, pedido, inicio de sesion, etc.

El selector de idioma es necesario para páginas multilenguaje.

Resumen

Se explicaron los elementos que se pueden incluir en el header, al igual que algunas temáticas que requieren elementos en específico.
